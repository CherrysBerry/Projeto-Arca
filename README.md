Esse projeto busca dar ideias para o site do projeto ARCA da Prefeitura da Serra - ES. Foi feito um site de ideia para o projeto ARCA, a qual aprendemos habilidades como o Figma, PostgreSQL, html, css e javascript, inclusive nesse projeto foi feito uma CRUD. Foi desenvolvido por Leandro Kuster, Luis Guilherme, Milena Wasconcellos, Rafael Pavani e Sophia Caliari, para a finalização do 1° período de Ciência da Computação na UVV.
